# Manus Prompt: AIKINTEL UI Alignment

You are designing the UI for the Crypto Edge AI module.

## Goal

Create a `Crypto Edge AI` experience that can later feel native inside AIKINTEL and supports trader-facing decision-support views.

## Required Sections

- Overview.
- Research Review.
- New Token Scanner.
- Risk Engine.
- Setup Review.
- Final Checklist.
- Projects.
- Opportunities.
- Scam Alerts.
- On-Chain.
- Market Summary.
- Optional Crypto Edge panel.

## Required Data Display

- Score 0-100.
- Sentiment/bias.
- Confidence 0-100.
- Risk factors.
- Checklist.
- Key points.
- Research summary.
- Disclaimer.

## Design Rules

- Use React 19.
- Use Tailwind CSS 4.
- Use shadcn/ui.
- Use wouter route conventions.
- Use TanStack Query through tRPC.
- Use dark AIKINTEL theme.
- Use cards, tabs, badges, skeletons, and tooltips.
- Keep the page dense, readable, and operational.

## Safety Rules

- Do not show buy/sell CTAs.
- Do not use profit-promise language.
- Do not imply automated execution.
- Do not add exchange, MT4, Telegram, Discord, or payment UI.
